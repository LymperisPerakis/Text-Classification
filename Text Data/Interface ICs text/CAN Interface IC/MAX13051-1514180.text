The MAX13051 ±80V fault-protected CAN transceiver         o Fully Compatible with the ISO 11898 Standard
with autobaud is ideal for device net and other industri-
al network applications where overvoltage protection is   o Autobaud Mode
required. The MAX13051 provides a link between the        o Short-Circuit Protection
CAN protocol controller and the physical wires of the
bus lines in a control area network (CAN).                o High-Speed Operation Up to 1Mbps
The MAX13051 features three different modes of opera-     o Slope-Control Mode
tion: high speed, slope control, and standby. High-       o Low-Current Standby Mode
speed mode allows data rates up to 1Mbps. The
slope-control mode can be used to program the slew        o Thermal Shutdown
rate of the transmitter for data rates of up to 500kbps,  o Transmit Data Dominant Timeout
reducing the effects of EMI and allowing the use of
unshielded-twisted or parallel cable. In standby mode,    o ±6kV Human Body Model ESD Protection
the transmitter shuts off and a low-power receiver moni-  o Greater than ±12V Common-Mode Range
tors the bus, waiting for a wake-up signal.

nant timeout function that prevents erroneous CAN con-
trollers from clamping the bus to a dominant level if the
