n  Protected from Overvoltage Line Faults to ±60V                         The LTC®2875 is a robust high speed, low power CAN
n  3.3V or 5V Supply Voltage                                              transceiver operating on 3.3V or 5V supplies that features
n  High Speed CAN FD Operation Up to 4Mbps                                ±60V overvoltage fault protection on the data transmis-
n  ±25kV ESD Interface Pins, ±8kV All Other Pins                          sion lines during all modes of operation, including power-
n  Variable Slew Rate Driver with Active Symmetry                         down. The maximum data rate has been extended to
   Control and SPLIT Pin for Low Electromagnetic                          4Mbps to support high speed protocols based on the CAN
   Emission (EME)                                                         physical layer. Supports up to 4Mbps CAN with Flexible
n  Extended Common Mode Range (±36V)                                      Data Rate (CAN FD). Enhanced ESD protection allows
n  Ideal Passive Behavior to CAN Bus with Supply Off                      these parts to withstand ±25kV HBM on the transceiver
n  Current Limited Drivers and Thermal Shutdown                           interface pins without latchup or damage.
n  Power-Up/Down Glitch-Free Driver Outputs                               Extended ±36V input common mode range and high
n  Micropower Shutdown Mode                                               common mode rejection on the CAN receiver provides
n  Transmit Data (TXD) Dominant Timeout Function                          tolerance of large ground loop voltages. A sophisticated
n  ISO 11898-2 and CAN FD Compliant
                                                                          CAN driver with active symmetry control maintains tight
n  DeviceNet Compatible
                                                                          control of the common mode voltage for excellent elec-
n  Up to MP-Grade Available (–55°C to 125°C)                              tromagnetic emission, while the variable slew rate and
n  3mm × 3mm 8-Lead DFN and SO-8 Packages                                 split termination support allow additional EME reduction.
