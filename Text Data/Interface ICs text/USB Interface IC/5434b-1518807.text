Chapter 2 Overview
The SMSC USB5434B hub is a 4-port, low-power Hub Controller fully compliant with the USB 3.0 Specification [2].
The USB5434B supports 5 Gbps SuperSpeed (SS), 480 Mbps Hi-Speed (HS), 12 Mbps Full-Speed (FS) and
1.5 Mbps Low-Speed (LS) USB signalling for complete coverage of all defined USB operating speeds.
All required resistors on the USB ports are integrated into the hub. This includes all series termination resistors and
all required pull-down and pull-up resistors on D+ and D- pins. The over-current sense inputs for the downstream
facing ports have internal pull-up resistors.
The USB5434B includes MultiTRAKTM technology, which implements a dedicated Transaction Translator (TT) for
each port. Dedicated TTs help maintain consistent full-speed data throughput regardless of the number of active
downstream connections.
The hub controller provides a default configuration, expediting implementation.
