FEATURES                                                                         DESCRIPTION
nn  One RS485 or One RS232 Transceiver                                           The LTC®2873 is a robust pin-configurable multiprotocol
nn 3V to 5.5V Supply Voltage                                                     transceiver that supports RS232, RS485, and RS422
nn Up to 20Mbps RS485                                                            protocols while operating on a single 3V to 5.5V supply.
nn Slew-Controlled RS232 Operation:                                              The LTC2873 can be configured as a half-duplex RS485
    nn Selectable 1Mbps or 250kbps                                               transceiver or as an RS232 transceiver using the same
nn Automatic Selection of Integrated RS485 (120Ω)                                two bus pins.
    and RS232 (5kΩ) Termination Resistors
nn High ESD: ±26kV HBM
                                                                                 A pin-controlled integrated termination resistor allows for
nn Logic Loopback Mode
                                                                                 easy interface reconfiguration, eliminating external resis-
nn 1.7V to 5.5V Logic Interface
                                                                                 tors and control relays. Loopback mode steers the driver
                                                                                 inputs to the receiver outputs for diagnostic self-test.
nn Supports Up to 256 RS485 Nodes
nn RS485 Receiver Failsafe Eliminates UART Lockup
                                                                                 The RS485 receiver supports up to 256 nodes per bus,
nn H-Grade Available (–40°C to 125°C)
                                                                                 and features full failsafe operation for floating, shorted or
nn Available in 24-Pin 4mm × 5mm QFN Package
                                                                                 terminated inputs.
                                                                                 An integrated DC/DC boost converter uses a tiny
                                                                                 2mm × 1.6mm inductor and one capacitor, eliminating
APPLICATIONS                                                                     the need for multiple supplies when driving RS232 levels.

nn  Industrial Sensors and Actuators
nn Alarm Systems
nn Traffic Control and Monitoring
nn Highway Signs and Jumbo Displays
