■   Operates from ±5V to ±15V Supplies                                           The LT ®1039 is a triple RS232 driver/receiver that in-
■   Fully Protected Against Overload                                             cludes shutdown. Each receiver will accept up to ±30V
■   Outputs Can Be Driven ±30V Without Damage                                    input and can drive either TTL or CMOS logic. The RS232
■   Three-State Outputs; Outputs Open When Off                                   drivers accept TTL logic inputs and output RS232 voltage
■   Bipolar Circuit—No Latchup                                                   levels. The outputs are fully protected against overload
■   ±30V Input Range                                                             and can be shorted to ground or up to ±30V without
■   Triple Driver/Receiver                                                       damage to the drivers. Additionally, when the system is
■   No Supply Current in Shutdown                                                shut down or power is off, the outputs are in a high
■   30kΩ Input Impedance                                                         impedance state allowing data line sharing. Bipolar cir-
■   Meets All RS232 Specifications                                               cuitry makes this driver/receiver exceptionally rugged
■   16-Pin Version—Pin Compatible with MC145406                                  against overloads and immune to latchup.
■   Available in SO Package                                                      A BIAS pin allows one receiver to be kept on while the rest
■   15kV ESD: LT1039A                                                            of the part is shut down.
