FEATURES                                                                                            DESCRIPTION
n   RS232 Transceiver: 2500VRMS for 1 Minute                                                              The LTM®2882 is a complete galvanically isolated dual
n   UL-CSA Recognized           File #E151738                                                             RS232 µModule® (micromodule) transceiver. No external
n   CSA Component Acceptance Notice 5A                                                                    components are required. A single 3.3V or 5V supply
n   Isolated DC Power: 5V at Up to 200mA                                                                  powers both sides of the interface through an integrated,
n   No External Components Required                                                                       isolated DC/DC converter. A logic supply pin allows easy
n   1.62V to 5.5V Logic Supply for Flexible Digital                                                       interfacing with different logic levels from 1.62V to 5.5V,
    Interfacing                                                                                           independent of the main supply.
n   High Speed Operation
                                                                                                          Coupled inductors and an isolation power transformer
       1Mbps for 250pF/3kΩ Load
                                                                                                          provide 2500VRMS of isolation between the line transceiver
       250kbps for 1nF/3kΩ Load                                                                           and the logic interface. This device is ideal for systems
       100kbps for 2.5nF/3kΩ TIA/EIA-232-F Load                                                           with different grounds, allowing for large common mode
n   3.3V (LTM2882-3) or 5V (LTM2882-5) Operation
                                                                                                          voltages. Uninterrupted communication is guaranteed for
n   No Damage or Latchup to ±10kV HBM ESD on
                                                                                                          common mode transients greater than 30kV/μs.
    Isolated RS232 Interface or Across Isolation Barrier
n   High Common Mode Transient Immunity: 30kV/μs                                                          This part is compatible with the TIA/EIA-232-F standard.
n   Maximum Continuous Working Voltage: 560VPEAK                                                          Driver outputs are protected from overload and can be
n   True RS232 Compliant Output Levels                                                                    shorted to ground or up to ±15V without damage. An
n   15mm × 11.25mm BGA and LGA Packages                                                                   auxiliary isolated digital channel is available. This channel
                                                                                                          allows configuration for half-duplex operation by control-
APPLICATIONS                                                                                              ling the DE pin.
n   Isolated RS232 Interface                                                                              Enhanced ESD protection allows this part to withstand up
n   Industrial Communication                                                                              to ±10kV (human body model) on the transceiver interface
n   Test and Measurement Equipment                                                                        pins to isolated supplies and across the isolation barrier
n   Breaking RS232 Ground Loops                                                                           to logic supplies without latchup or damage.
