Features                                                                              Description
n On-Chip ESD Protection:                                                             The LTC®1039A is a triple RS232 driver/receiver that
		±15kV Human Body Model                                                              includes shutdown. Each receiver will accept up to ±30V
		±15kV IEC-1000-4-2 Air Gap Test**                                                   input and can drive either TTL or CMOS logic. The RS232
		±8kV IEC-1000-4-2 Contact Test                                                      drivers accept TTL logic inputs and output RS232 voltage
n 125kBd Operation with 3kΩ/2500pF Load                                               levels. The outputs are fully protected against overload and
n 250kBd Operation with 3kΩ/1000pF Load                                               can be shorted to ground or up to ±30V without damage
n Outputs Can Be Driven ±30V Without Damage                                           to the drivers. Additionally, when the system is shut down
n No Supply Current in Shutdown                                                       or power is off, the outputs are in a high impedance state
n Operates from ±5V to ±15V Supplies                                                  allowing data line sharing. On-chip ESD protection elimi-
n Fully Protected Against Overload                                                    nates the need for external protection devices.
n Three-State Outputs; Outputs Open When Off
n Bipolar Circuit—No Latchup
                                                                                      A bias pin allows one receiver to be kept on while the rest
n ±30V Input Range
                                                                                      of the part is shut down.
n Triple Driver/Receiver                                                              The LT1039A is also available in the 16-pin version, without
n 30kΩ Input Impedance                                                                shutdown or bias pin functions.
n Meets All RS232 Specifications
                                                                                      For applications requiring operation from a single 5V
n 16-Pin Version—Pin Compatible with MC145406
