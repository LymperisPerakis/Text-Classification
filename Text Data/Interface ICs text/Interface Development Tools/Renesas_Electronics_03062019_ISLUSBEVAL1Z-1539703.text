1.    Overview
The Renesas USB to PMBus Adapter (often referred to as a ‚Äúdongle‚Äù) is used to connect a demonstration board
with a PMBus interface to a PC. The USB to PMBus adapter is powered from the USB output of the host PC. The
USB interface utilizes the USB Mini format, and the output uses a standard 2 row, 6 pin header on 0.100 inch
centers. The PMBus command set is accessed by using the PowerNavigator‚Ñ¢ evaluation software from a PC
running Microsoft Windows. PowerNavigator software is downloaded from the Renesas website.
