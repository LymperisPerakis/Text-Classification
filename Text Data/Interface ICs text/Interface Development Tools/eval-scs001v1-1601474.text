                                                                  Fast and easy migration from DC barrel to Type-C
                                                        Features
                                                        •     USB Power Delivery SINK port
                                                        •     Short-to VBUS protection up to 28 V
                                                        •     Low BOM cost, small footprint
                                                        •     Status LED
                                                        Description
                                                        The EVAL-SCS001V1 reference design allows user to create a USB Type-C
                                                        connector quickly and easily in order to power any application up to 100 W (20 V, 5
                                                        A).
