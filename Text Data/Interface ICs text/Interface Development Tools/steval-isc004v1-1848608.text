                                                                 STEVAL-ISC004V1
                  STUSB4710A evaluation board (with on-board DC-DC)
                                                                                                Data brief
                                                       Description
                                                       The STEVAL-ISC004V1 evaluation board is a
                                                       ready-to-use USB PD source based on
                                                       STUSB4710A and ST1S14 ICs. It demonstrates
                                                       how to convert a fixed voltage DC power input
                                                       into a USB PD variable voltage output.
                                                       The STUSB4710A acts as a Type-C port
                                                       interface that ensures power delivery advertising
                                                       and negotiation, drives the DC-to-DC converter
                                                       and power paths, and monitors VBUS voltage.
                                                       The ST1S14 device manages the step down
                                                       conversion from the DC input port to the
                                                       negotiated VBUS voltage.
Features                                               The various LEDs indicate the operating status of
     1 x USB PD port (source)                         the STUSB4710A and USB PD port.
     STUSB4710A USB power delivery controller
     Compliant with:                                  The USB PD port is pre-configured with five
          USB Type-C r1.2                             different PDOs to address a broad range of
                                                       applications (5 V, 9 V, 12 V, 15 V and 20 V).
          USB PD r2.0
     On-board DC-DC
     VBUS power switches and discharge path
     Support for up to 5 power data objects
      (PDO)
     Short-to-VBUS protections on CC pins (22 V)
      and VBUS pins (28 V)
     Customizable start-up profiles
     Compatible with NUCLEO-F072RB board for
      configuration and debug interface
     RoHS compliant
