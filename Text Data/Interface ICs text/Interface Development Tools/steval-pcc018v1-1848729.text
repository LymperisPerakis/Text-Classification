                                                                                      STEVAL-PCC018V1
                                                                                              UART to USB bridge
                                                                                                                    Data brief
                                                                            Description
                                                                            Based on the Silab CP2102 USB bridge, the
                                                                            STEVAL-PCC018V1 is a USB to UART board that
                                                                            allows the integration of a USB connector with a
                                                                            general-purpose board which has a UART
                                                                            interface available.
Features
• USB and UART interfaces
• Powered via USB
• Integrated USB-to-serial bridge
• Integrated voltage converter
• Integrated power LED
• Dimensions: 17 mm x 42 mm
• RoHS compliant
