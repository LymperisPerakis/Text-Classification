ply. The MXL1543, along with the MXL1544/MAX3175                                                                                                       Chipset Is Pin Compatible with LTC1543,

software-selectable resistor termination network or by                                                                                                 MAX3175, and MXL1344A
discrete termination networks.                                                                                                                       ♦ +5V Single-Supply Operation
