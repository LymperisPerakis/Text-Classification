General Description                                            Features
The MAX6674 cold-junction-compensation thermocou-              ●● Cold-Junction Compensation
ple-to-digital converter performs cold-junction compensa-      ●● Simple SPI-Compatible Serial Interface
tion and digitizes the signal from a type-K thermocouple.
The data is output in a 10-bit resolution, SPI™-compatible,    ●● 10 Bit, 0.125°C
read-only format.                                              ●● Open Thermocouple Detection
This converter resolves temperatures to 0.125°C, allows
readings as high as +128°C, and exhibits thermocouple
accuracy of ±2°C for temperatures ranging from 0°C to
+125°C.                                                        Ordering Information
