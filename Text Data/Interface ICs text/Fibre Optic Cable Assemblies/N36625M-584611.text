Description
                                                                                                      Singlemode Fiber Patch Cable,
Tripp Lite's 25-meter ( 82ft ) singlemode duplex fiber optic LC/SC patch cable is manufactured from
                                                                                                      LC/SC
8.3/125 zipcord fiber. The cable has SC connectors on one end, LC connectors on the other, a PVC
jacket, and is FDDI and OFNR rated. Duplex singlemode fiber is most commonly used in LAN
applications.
Features
●  Manufactured from 8.3/125 duplex (zipcord) fiber
●  PVC jacket
●  Length: 25 meters ( 82ft ). Connectors: 2 x SC; 2 x LC
●  Insertion loss testing performed on every connector (0.2db typical) and provided with cable
●  Fiber made from glass (not a polymer)
●  Fiber optic distributed data interface (FDDI) rated
●  OFNR (riser rated)
