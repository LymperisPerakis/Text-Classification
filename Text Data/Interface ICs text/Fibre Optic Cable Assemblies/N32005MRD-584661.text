Description
                                                                                                            62.5/125 Micron Fiber Red
Tripp Lite's Red, 5-meter multimode duplex micron fiber optic LC/LC patch cable is manufactured from
                                                                                                            Jacket
62.5/125 zipcord fiber. The cable has LC connectors on each end, a PVC jacket, and is FDDI and OFNR
rated. Duplex multimode fiber is most commonly used in LAN applications. Ideal for color coding the cable
plant.
Features
