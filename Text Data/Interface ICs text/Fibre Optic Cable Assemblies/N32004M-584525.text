Description
Tripp Lite's 4-meter ( 13ft ) multimode duplex micron fiber optic LC/LC patch cable is manufactured from
62.5/125 zipcord fiber. The cable has LC connectors on each end, a PVC jacket and is FDDI and OFNR
rated. Duplex multimode fiber is most commonly used in LAN applications.
Features
●  Manufactured from 62.5/125 duplex (zipcord) fiber
●  PVC jacket
●  Length: 4 meters ( 13ft ) Connectors: LC connector on each end
●  Insertion loss testing performed on every connector (0.2db typical) and provided with cable
●  Beveled edge on ends of glass makes insertion of plug a breeze
●  Fiber made from glass (not a polymer)
●  Fiber optic distributed data interface (FDDI) rated
●  OFNR (riser rated)
