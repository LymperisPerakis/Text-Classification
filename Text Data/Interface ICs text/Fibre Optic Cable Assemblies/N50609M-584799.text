Description                                                                                                  cable with SC/SC connectors
Tripp Lite's 9-meter multimode duplex Fibre Channel SC/SC patch cable is manufactured from 50/125
zipcord fiber. The cable has SC connectors on each end, a PVC jacket and is FDDI and OFNR rated.
